介绍

Pandas是基于Numpy构建的库，在数据处理方面可以把它理解为numpy加强版。不同于numpy的是，pandas拥有种数据结构：Series和DataFrame：

用代码生成一个简单的Series对象

data = {‘a’:[1,2,3],’b’:[‘we’,’you’,’they’],’c’:[‘btc’,’eos’,’ae’]}
df = DataFrame(data)
df
a b c
0 1 we btc
1 2 you eos
2 3 they ae

data = Series([222,’btc’,234,’eos’])
data
0 222
1 btc
2 234
3 eos
dtype: object

data = Series([1,2,3,4],index = [‘a’,’b’,’c’,’d’])
data
a 1
b 2
c 3
d 4
dtype: int64

dic = {‘a’:1,’b’:2,’c’:’as’}
dicSeries = Series(dic)
dicSeries

b 2
c as
dtype: object

data.index
Index([‘a’, ‘b’, ‘c’, ‘d’], dtype=’object’)
data.values
