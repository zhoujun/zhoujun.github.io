简介

Flask-APScheduler是Flask扩展，增加了对APScheduler的支持。

  • 支持从Flask配置加载调度程序配置以及任务定义。
  • 支持WEB程序和调度程序共存。提供REST API管理任务调度。并为REST API提供身份验证。目前只支持Basic-Auth认证。

官方地址：https://github.com/viniciuschiele/flask-apscheduler
